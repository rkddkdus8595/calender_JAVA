### Calendar Memo
> 로그인을 통해 원하는 날짜를 선택하여 메모를 입력하는 프로젝트 <br>
> A project to select a desired date through login and enter a memo

### Use Technique (사용 기술)
> JAVA, MSSQL

### DB
> id, password, name, phone (Join Table) <br>
> Year, Month, Day, Memo, id (Calendar Memo Table)

### Team Project (My Function)
> Design + Calendar + Memo
> 디자인 + 달력 + 메모 (메모 관련 폰트 수정 기능 등등)
